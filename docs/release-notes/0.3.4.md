# @gaozh1024/rn-kit 0.3.4 Release Notes

`0.3.4` 是一个状态栏体验与文档同步修复版本，重点让带 `AppHeader` 的页面在有色头部场景下获得更自然的顶部显示效果，同时保持全局状态栏默认行为不变。

## 修复内容

### 1. `AppHeader` 自动处理聚焦态透明状态栏

- `AppHeader` 内部自动注入 `AppFocusedStatusBar`
- 配置为：
  - `translucent`
  - `backgroundColor="transparent"`
- 带 `AppHeader` 的页面顶部状态栏区域会直接显示 Header 自身背景色

适合场景：

- 设置页
- 二级详情页
- 使用品牌色 / 卡片色 Header 的页面

### 2. 保持 `AppStatusBar` 全局默认语义

- `AppStatusBar` 默认仍为 `translucent={false}`
- 全局默认背景色继续跟随当前主题背景色
- 避免把沉浸式状态栏行为扩散到所有未显式处理顶部安全区的页面

### 3. 文档同步更新

已同步更新：

- `packages/framework/README.md`
- `SETUP.md`
- `templates/expo-starter/README.md`
- `docs/README.md`

文档现在明确说明：

- `AppHeader` 会自动处理 Header 场景下的透明状态栏
- 全屏彩色 / 渐变页面推荐使用 `AppFocusedStatusBar`
- 全局默认状态栏仍保持非沉浸式

## 配套版本

- `@gaozh1024/rn-kit`：`0.3.4`
- `@gaozh1024/expo-starter`：`0.2.3`（依赖 `@gaozh1024/rn-kit@^0.3.4`）
