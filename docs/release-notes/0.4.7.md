# @gaozh1024/rn-kit 0.4.7 Release Notes

`0.4.7` 是一次页面容器与输入体验修正发布，重点解决 `AppHeader` 顶部安全区职责、状态栏切换闪动，以及 `AppInput` 高度控制问题。

## 本次更新

### 1. 统一 Header 页顶部安全区规则

- `AppScreen` 继续默认 `top=false`
- `AppHeader` 默认 `safeArea=true`
- 推荐约定明确为：
  - **Header 页**：顶部安全区交给 `AppHeader`
  - **非 Header 页**：使用 `AppScreen top` 或 `SafeScreen`

这样 `status bar + header` 会保持同一块背景，避免顶部颜色割裂。

### 2. 调整状态栏默认策略，降低首屏闪动

- `AppStatusBar` 默认改为 `translucent=true`
- 减少全局状态栏与页面聚焦态状态栏切换时的顶部布局跳动

### 3. 修复 `AppInput` 高度控制

- `style.height` 现在会作用到输入框外层容器
- 新增：
  - `containerStyle`
  - `inputStyle`

适合分别控制整体高度和文本样式。

### 4. 文档与模板说明同步更新

- 框架 README 补充 Screen / Header 推荐规则
- 模板 README 与 `PageScreen` 示例同步更新

### 5. 补齐 `AppPressable` / `AppText` 快捷参数

- `AppPressable` 现在支持与基础容器一致的常用快捷参数
  - 布局：`flex` / `row` / `wrap` / `center` / `between` / `items` / `justify` / `gap`
  - 间距：`p` / `px` / `py` / `pt` / `pb` / `pl` / `pr` / `m` / `mx` / `my` / `mt` / `mb` / `ml` / `mr`
  - 尺寸：`w` / `h` / `minW` / `minH` / `maxW` / `maxH`
  - 外观：`rounded` / `bg` / `surface`
- `AppText` 增加文本场景常用快捷参数
  - 支持：`flex` / 间距 / 尺寸 / `rounded` / `bg` / `surface`
  - 不支持：`row` / `gap` / `items` / `justify` 等容器语义参数

### 6. 第一批业务组件补齐基础快捷参数

- `GradientView`：支持布局 / 间距 / 尺寸 / `rounded`
- `AppButton`：支持 `flex` / 外边距 / 尺寸 / `rounded`
- `AppInput`：支持外层排版快捷参数，输入容器支持尺寸 / `rounded` / `bg` / `surface`
- `Select` / `Picker` / `DatePicker`：支持外层排版快捷参数，触发器支持尺寸 / `rounded` / `bg` / `surface`
- `FormItem`：支持布局 / 间距 / 尺寸 / `bg` / `surface` / `rounded`

### 7. 第二批业务组件补齐基础快捷参数

- `Checkbox` / `Radio`：支持外层常用排版快捷参数，适合直接控制点击热区、标签间距与尺寸
- `Switch`：支持外层基础排版参数，轨道支持 `rounded`
- `Slider`：支持外层宽度/外边距快捷参数，轨道支持 `rounded` / `bg` / `surface`
- `CheckboxGroup` / `RadioGroup`：组容器支持与 `AppView` 接近的一套布局快捷参数
- `Progress`：支持外层尺寸 / 外边距，轨道支持 `rounded` / `bg` / `surface`

### 8. 第三批快捷参数与组件清理

- `AppList`：支持与 `AppFlatList` 接近的一套基础快捷参数
- `AppImage`：保留 `width/height/borderRadius` 的同时，新增 `w/h/rounded` 等快捷别名
- `Icon`：支持基础外层快捷参数，便于图标按钮、状态入口快速排版
- 清理未对外导出的旧版 `ui/feedback/Progress.tsx`，避免维护分叉
