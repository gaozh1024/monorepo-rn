# @gaozh1024/expo-starter 0.2.8 Release Notes

`0.2.8` 是一次模板跟进发布，主要同步依赖到 `@gaozh1024/rn-kit@0.4.3`，用于承接本轮框架体验修正。

## 本次版本说明

- 模板依赖升级到 `@gaozh1024/rn-kit@^0.4.3`
- 模板内置版本号同步为 `0.2.8`
- 保持现有开发态可观测性、错误边界与 API observability 默认配置
